# مقر جمعية طويق لصناعة الكوادر البشرية

موقع عربي تعريفي من صفحة واحدة لمشروع مقر جمعية طويق لصناعة الكوادر البشرية.

## النشر على GitHub Pages

1. ارفع محتويات هذا المجلد إلى مستودع GitHub.
2. من إعدادات المستودع اختر `Pages`.
3. اختر النشر من فرع `main` ومجلد `/root`.
4. الملف الرئيسي هو `index.html`.

## البنية

- `index.html`: محتوى الصفحة.
- `styles.css`: التصميم والاستجابة للجوال.
- `script.js`: القائمة، تفعيل التنقل، وحركات الظهور.
- `assets/images`: صور المشروع المحسنة بصيغة WebP.
- `assets/logos`: الشعارات الرسمية المستخدمة في الموقع.
